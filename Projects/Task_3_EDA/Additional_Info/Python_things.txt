fillna(): A pandas function used to fill missing values in a Dataframe or series
Example:
df['column_name'].fillna(value)

mode(): A pandas method that calculates the most frequent value in a column. It returns a Series of the most frequent value(s), and it can be used on both numerical and categorial data.
mode()[0]: it returns the first value in that series

inplace = True: is a parameter that modifies the original object(like the dataframe or series) directly, without needing to reassign it.
i.e. modifies the df dataframe in-place (without creating a new copy)
